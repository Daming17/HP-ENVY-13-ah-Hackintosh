# HP-ENVY-13-ah1002tx-Hackintosh

HP ENVY 13 ah1002tx Hackintosh EFI

![HP-ENVY-13-ah1002tx](./images/HP-ENVY-13-ah1002tx.png)

## 配置信息

- CPU: Intel Core i5-8265U (Whiskey Lake)
- GPU: Intel UHD Graphics 620
- Mobo: HP 850B (Intel 300 Series Chipset)
- Wifi & Bluetooth: Intel Wireless-AC 9560 160MHz
- Audio: Realtek ALC285

## 说明

当前使用的 `OpenCore` 版本为 `v1.0.2 Debug`，`macOS` 版本为 `Sonoma 14.7.1`；安装时使用在线安装方式安装，在 `Sequoia` 的恢复模式下使用 `AirportItlwm v2.3.0` 无法驱动无线网卡，因此使用 `Sonama` 安装，`Sequoia` 下的网卡驱动暂未测试。
